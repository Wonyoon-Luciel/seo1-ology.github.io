# Seowon Yoon - Security Engineer Portfolio

보안 엔지니어 윤서원(Seowon Yoon)의 개인 포트폴리오 웹사이트입니다.
이 프로젝트는 저의 경력, 진행한 프로젝트, 그리고 연락처 정보를 소개하기 위해 제작되었습니다.

## 📌 주요 기능 (Features)
- **Home**: 첫 인상을 주는 인트로 화면과 네비게이션
- **About**: 자기소개 및 보유 기술 스택(Skills)
- **My Work**: 주요 프로젝트 및 연구 성과 소개
- **Contact**: 연락처 및 GitHub 링크 제공
- **Responsive Design**: 데스크탑 및 모바일 환경 지원

## 🛠 기술 스택 (Tech Stack)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS + jQuery)
- **Design**: Minimalist & Modern UI (Custom CSS)

## 📂 프로젝트 구조 (Structure)
```
portfolio/
├── index.html      # 메인 페이지 (Home)
├── about.html      # 소개 페이지 (About)
├── mywork.html     # 프로젝트 페이지 (My Work)
├── contact.html    # 연락처 페이지 (Contact)
├── assets/         # CSS, JS, Fonts 등 정적 자원
└── images/         # 프로필 및 프로젝트 이미지
```

## 🚀 실행 방법 (How to Run)
이 프로젝트는 정적 웹사이트이므로 별도의 서버 설정 없이 바로 실행할 수 있습니다.

1. 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 브라우저(Chrome, Safari 등)에서 엽니다.
